### 特征回溯项目的字段批量填写模式

> 三方数据特征回溯项目通常涉及数百甚至数千个字段，
> 不可能逐个按12列完整填写。可采用以下批量填写方式：

**批量填写格式**：

```
三方产品：[产品名]
产品code：[code]
字段数量：[N个]

| 字段名称 | 数据类型 | 取值范围 | 默认值 | 备注 |
|---|---|---|---|---|
| [特征1名] | int | [0,100] | -1 | |
| [特征2名] | varchar | [枚举值列表] | -2 | |
| ... | ... | ... | ... | |

公共计算逻辑：通过uid+产品code异步批量调用三方接口获取
公共数据源说明：[产品接口名]（需实时）
公共查询参数：uid + order_id
```

## 五、默认值约定

| 数据类型 | 查无数据 | 计算异常 | 说明 |
|---|---|---|---|
| int | -1 | -2 | -1=查无数据，-2=计算异常 |
| varchar | "" | -2 | 空字符串=查无数据，"-2"=计算异常 |
| double | -1.0 | -2.0 | -1.0=查无数据，-2.0=计算异常 |

> **重要**：默认值约定需与研发团队确认一致。如项目有特殊约定，需在此标注。
