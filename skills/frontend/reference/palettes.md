## CSS Token 映射

色板 7 字段 → CSS 变量映射（Step 2.7 设计简报输出时使用）：

| 色板字段 | CSS 变量 |
|---------|---------|
| 主色 | `--color-primary` |
| 背景 | `--color-bg` |
| 卡片 | `--color-surface` |
| 文字主 | `--color-text` |
| 文字辅 | `--color-text-muted` |
| 边框 | `--color-border` |

## 场景推荐映射

### Dashboard 候选
优先：蓝钢(1) > 墨绿(2) > 暗黑(5) > 石墨(8)

### Landing 人格映射

| 人格 | 推荐色板 | 主色 |
|------|---------|------|
| 极简克制 | 石墨(8) 或 深靛(3) | 黑白灰+单强调色 |
| 温暖手工 | 暖土(4) | 暖橙/陶土 |
| 科技前卫 | 深靛(3) 或 海蓝(7) | 饱和蓝紫 |
| 经典专业 | 海蓝(7) 或 墨绿(2) | 深蓝/墨绿 |
| 创意大胆 | 玫红(6) | 玫红/粉 |

### Creative 风格 → 默认色板

| 风格 | 默认色板 | 备选 |
|------|:--:|------|
| 极简白 | 石墨(8) | 蓝钢(1)、深靛(3) |
| 粗野主义 | 暗黑(5) | 石墨(8)、玫红(6) |
| 编辑式 | 暖土(4) | 海蓝(7)、石墨(8) |
| 暗色沉浸 | 暗黑(5) | 深靛(3)、玫红(6) |
| 复古暖色 | 暖土(4) | 玫红(6)、墨绿(2) |
| 实验性 | 玫红(6) | 暗黑(5)、深靛(3) |
