# 风控常见模块

> 本文件列出风控项目常见功能模块和组件，供生成PRD时参考选用。
> 根据项目实际需要选择/增减，不是所有项目都需要全部模块。

---

## 模块分类

### 一、决策引擎相关模块

| 模块 | 功能说明 | 适用项目类型 |
|---|---|---|
| 条件规则配置 | 配置基于条件判断的规则（如：年龄>18 且 信用分>600 → 通过） | 规则配置类、Agent类 |
| 脚本规则配置 | 配置基于脚本/表达式计算的规则 | 规则配置类、特征计算类 |
| 规则上线/下线 | 规则部署到决策引擎或从引擎撤回 | 规则管理类、Agent类 |
| 规则效果追踪 | 记录规则上线后的命中率、误杀率、GMV影响 | 规则管理类、Agent类 |
| 规则唯一标记 | 每条规则打上唯一ID标记，关联线上策略 | Agent类、规则追踪类 |
| 规则冲突检测 | 检测新增规则与存量规则的冲突 | 规则管理类 |

### 二、智能体/对话相关模块

| 模块 | 功能说明 | 适用项目类型 |
|---|---|---|
| 对话输入框 | 用户输入自然语言需求的文本输入区域 | Agent类、回捞类 |
| 对话历史管理 | 管理多个对话任务、历史记录、任务ID | Agent类、回捞类 |
| 话术模板库 | 预设的机器人回复话术模板 | Agent类、回捞类 |
| 话术审核机制 | 话术模板上线前的审核流程 | Agent类、回捞类 |
| 对话辅助按钮 | 复制/重新编辑/重新生成/点赞/点踩 | Agent类、回捞类 |
| 文件上传 | 支持用户上传Excel/CSV/TXT等补充材料 | Agent类、回捞类 |
| 能力卡片区 | 展示智能体核心能力的入口卡片 | Agent类 |
| 规则配置跳转 | 从对话跳转到米霍克规则配置页 | Agent类 |
| IM对话通道 | 企微/IM对话通道接入 | 回捞类 |

### 三、数据/特征相关模块

| 模块 | 功能说明 | 适用项目类型 |
|---|---|---|
| 大宽表查询 | 查询包含基础/交易/行为/特征数据的宽表 | 数据类、策略分析类 |
| 特征回溯 | 异步批量回溯三方数据特征 | 数据类、特征类 |
| 特征空跑 | 新特征不参与决策但记录命中数据 | 数据类、特征类 |
| 数据旁路计算 | 异步拉取+计算+合并落库 | 数据类 |
| 样本构建 | 构建模型训练样本 | 模型类、策略类 |
| 特征推荐 | 推荐未使用但有价值的特征 | 数据类、策略类 |

### 四、评估/分析相关模块

| 模块 | 功能说明 | 适用项目类型 |
|---|---|---|
| 额度评估 | 评估客户限额并更新offer | 回捞类 |
| 策略挖掘 | 自动挖掘清退/准入规则 | Agent类（清退） |
| 策略效果评估 | 评估规则上线后的效果 | Agent类、规则管理类 |
| 策略分析报告 | 输出策略规则分析报告 | Agent类 |
| GMV损失控制 | 控制策略上线后的GMV损失在阈值内 | Agent类 |

### 五、流程/运营相关模块

| 模块 | 功能说明 | 适用项目类型 |
|---|---|---|
| 意愿确认 | 确认客户是否有意愿配合补件 | 回捞类 |
| 材料收集 | 收集客户补充的证明材料 | 回捞类 |
| 限额更新 | 更新客户借款限额 | 回捞类 |
| 引导借款 | 推送借钱卡片引导客户借款 | 回捞类 |
| 人工兜底 | 机器人异常时回退到人工处理 | Agent类、回捞类 |

## 模块组合参考（按项目类型）

### 清退策略Agent
> 大模型 + Agent平台 + 大数据平台 + 模型平台 + 米霍克决策引擎 + 规则配置跳转

### IM回捞智能体
> IM对话通道 + 话术模板库 + 意愿确认 + 材料收集 + 限额评估 + 引导借款 + 人工兜底

### 三方数据特征回溯
> 大宽表查询 + 特征回溯 + 数据旁路计算 + 特征推荐 + 特征空跑

### 规则管理平台
> 条件规则配置 + 脚本规则配置 + 规则上线/下线 + 规则效果追踪 + 规则冲突检测
